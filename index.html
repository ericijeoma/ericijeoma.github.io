Project Highlights

Multi-Agent Financial Intelligence Platform

LINCORNE TECHNOLOGIES · 2024 – present

Led a 6-person team to design and ship a production AI system where multiple specialized agents collaborate in a stateful LangGraph pipeline — each agent handling a distinct domain and handing tasks through MCP and A2A protocols with strict tool boundaries to eliminate data leakage between sessions.

Used RAG + NLP + SQL to ground every LLM response in structured financial databases, reducing hallucinations by 40% compared to the base model. The entire system is containerized with Docker and Kubernetes on AWS, exposed via REST APIs, and delivers streaming answers in under 500ms for simultaneous real users in live production. A full CI/CD pipeline with distributed tracing gives engineers complete visibility into agent behaviour at every step.
